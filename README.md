# Dashboard-WB05
* A Dashboard Visualization program (HTML & Google Sheets based)

## Run the Application
1. Clone this repository.
2. Create ```environment.js``` with these parameters
    ```js
    const ENV = {
        SHEET_ID: "YOUR_SHEET_ID"
        SHEET_NAME: "YOUR_SHEET_NAME"
    };
    ```
    (Raw html files can't read .env btw)
3. Open ```WebDashboard.html```
